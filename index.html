트레이너 포켓몬 포획

📋 사용 방법

  1. 트레이너와 배틀 진입
  2. 에뮬레이터에서 0xD0C8-0xD0CB 주소의 값 확인 (상대 기술)
  3. 아래 입력란에 확인한 값을 입력 (예: 0x5D)
  4. "생성" 버튼 클릭하여 메모리 테이블 확인
  5. 출력된 테이블을 에뮬레이터에 입력

상대 기술 입력

에뮬레이터에서 확인한 0xD0C8-0xD0CB 주소의 값을 입력하세요

⚡ 주요 변경사항

  • 0xD1D3: 전투 타입을 야생으로 변경 (0x01)
  • 0xCC1B-0xCC1E: 포획 시 기술 설정 (입력한 값으로 자동 설정)

⚠️ 이 기능은 게임 진행에 영향을 줄 수 있습니다. 사전에 세이브 파일을 백업하세요.
